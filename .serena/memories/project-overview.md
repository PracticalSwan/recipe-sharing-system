Project: Recipe Sharing System (React + Vite + Tailwind v4). Client-only app with localStorage persistence. Entry is in main render, Router is HashRouter. Roles: admin, user (contributor), pending guest; status controls permissions. Core features: auth, recipe CRUD, approval workflow, search/filter/sort, ratings/reviews, likes/favorites, profile editing, admin dashboards, daily stats/activity logs. LocalStorage is the single data layer; no backend API.
