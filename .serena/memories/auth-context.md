AuthContext: initializes storage, loads current user from localStorage, tracks loading. login(email,pw) -> storage.login; logout -> storage.logout and clear user. signup creates user with role user, status pending, then logs in. updateProfile merges and saves user. Derived flags: isAdmin, isPending, isSuspended, canInteract (active non-admin). Activity tracking: updateLastActive on intervals (1 min) and recordActiveUser hourly; also updates lastActive on unload/pagehide. Listens for favoriteToggled to sync current user.
