storage.js: localStorage keys for users, recipes, current user, guest id, reviews, search history, daily stats, activity. Seeds users (admins + users) and recipes with statuses published/pending/rejected. Core APIs: get/save users & recipes, delete user/recipe, login/logout, get/set current user, guest id, update lastActive, reviews CRUD, favorites/likes toggle, view tracking (user/guest), search history add/clear, stats (daily active/users/views), and activity log. recordView logs per-day view with viewerKey (user or guest). addActivity stores up to 200 items.
